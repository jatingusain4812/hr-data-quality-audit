HR Data Quality Audit & Risk Control System

Python | MySQL | SQL | Advanced Excel

Built an automated audit system to detect data quality issues in HR records (500+ employees)
Performed 11 audit checks including missing fields, invalid emails, negative salaries, and underage employee detection
Assigned weighted risk scores to each employee and classified them as High / Medium / Low / Clean risk
Wrote 12 MySQL queries for database-level auditing including duplicate detection and salary outlier analysis
Generated a 5-sheet Excel dashboard with KPI cards, risk summary, and department-wise analysis using Python (OpenPyXL)
Identified 294 data issues and flagged 42 high-risk records out of 500 employees
